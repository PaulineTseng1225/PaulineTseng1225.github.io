# Pauline Tseng — UX/UI Designer Portfolio

Hand-coded bilingual (EN / 繁中) portfolio site. No frameworks, no site builder.

**Live site:** https://paulinetseng1225.github.io

## How it's built
- Vanilla HTML / CSS / JavaScript
- All content lives in `assets/content.js` (bilingual data), rendered by `assets/render.js`
- Design tokens via CSS custom properties
- Custom CMS (`editor.html`) with live preview, versioning and autosave
- WCAG AA colour contrast; semantic HTML; keyboard accessible
